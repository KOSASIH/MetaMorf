MetaMorf Requirements

- pandas
- numpy
- matplotlib
- seaborn
- folium
- scikit-learn
- scipy
- tensorflow
- keras
- pytest
- coverage
- flake8
- black

This requirements.txt file lists the required packages and their versions for MetaMorf. The packages include pandas for data manipulation, numpy for numerical computations, matplotlib and seaborn for data visualization, folium for mapping, scikit-learn for machine learning, scipy for scientific computations, tensorflow and keras for deep learning, pytest and coverage for testing, flake8 for code linting, and black for code formatting.

To install the required packages, you can run the following command in the terminal:

```
pip install -r requirements.txt
```

This will install all the required packages and their dependencies in the virtual environment.

Note: The version numbers for the packages are not specified in this example. You can add the version numbers to the requirements.txt file if you want to lock the versions for your project. You can also specify the package names only, without the version numbers, if you want to install the latest versions of the packages.
